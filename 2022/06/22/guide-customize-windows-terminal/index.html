解决乱码问题

官方推荐下载 Meslo 字体.

注意,oh-my-posh使用的字体叫 Nerd.

下载了之后你会发现里面有很多的字体样式,如果不知道字体样式的区别.安装M开头字母的文件夹内所有字体就行了.各种字体的区别在Nerd的GIthub中有说明.

关于Visual Studio code

安装完后在VS code里面用Powershell做默认终端就有相应的效果了.

打开VS code终端时候应该还有部分的乱码,在设置里面找到终端字体进行修改.

有些字体的名字可能不会想想象的那么直观,就比如Meslo在系统的完整名字叫MesloLGMDZ Nerd Font.

可以先去terminal软件里面设置里先设置字体,然后在设置页面左下角找到打开JSON文件
